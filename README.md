######enum array tuple
```
enum E{A,B,C}  //0,1,2
enum E{A=1,B,C}  //1,2,3
enum E{A=5,B=4,C=1}  //5,4,1
```
tuple: first few elem: ```let x:[number,string]=[1,'x'];  ```
then we can add more elem (string,number only)
x[2]=1

######Duck typing
```
interface pro{
    walk:()=>void
}
let ins = {
    walk:()=>{console.log('walk');
}

function i(p:pro){}
i(ins);
```
